# ansible-playbooks
主にさくらVPSを使用した vagrant, ansible を用いた仮想環境のレシピファイルを公開していこうと思います。
## ホストOS

* OS: Windows 10 / Mac OS X (10.10)
* 仮想化ソフト: VirtualBox
* 仮想化ツール: Vagrant (1.9)

## ゲストOS

* OS: CentOS 6
* 構成管理ツール: Ansible
